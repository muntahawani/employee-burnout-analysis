# Employee Burnout Prediction

## 📌 Project Overview

This project analyzes employee burnout using machine learning.  
The objective is to identify key factors affecting burnout and build a predictive model to estimate burnout levels based on employee data.

---

## 📊 Dataset

The dataset contains features like:

- Gender  
- Company Type  
- WFH Setup Available  
- Designation  
- Resource Allocation  
- Mental Fatigue Score  
- Burn Rate (Target)

---

## ⚙️ Workflow

1. Data Loading  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Model Training  
5. Model Evaluation  

---

## 📈 Model Used

- Linear Regression

---

## 📊 Results

- R² Score: 0.86  
- The model performs well in predicting burnout levels.

---

## 🔍 Key Insights

- Mental Fatigue Score strongly affects burnout  
- Resource Allocation is another major factor
